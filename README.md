# Apollo Desktop

Apollo Desktop é um agente desktop responsável por receber e executar comandos remotos enviados por dispositivos clientes.

O projeto faz parte do ecossistema Apollo Workspace e possui foco em:

* comunicação em tempo real
* execução remota de comandos
* monitoramento de conexões
* arquitetura modular
* automação desktop

---

## Funcionalidades atuais

* Comunicação via socket
* Execução remota de comandos
* Abertura do Spotify
* Abertura do Google Chrome
* Abertura do explorador de arquivos
* Monitoramento de dispositivos conectados
* Histórico de comandos executados

---

## Tecnologias utilizadas

* Flutter Desktop
* Dart
* Socket Communication
